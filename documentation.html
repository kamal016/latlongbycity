City by lat-long API

CBLL offers a powerful, real-time Lat-long to geolocation API capable of looking up accurate location data of nearest city. Results are delivered within milliseconds in JSON format. Using the CBLL API you will be able to get nearest city data to map down in BI applications.
This documentation outlines in detail API features, available options and integration guide.

Getting Started

API Access Key

Your API Access Key is your unique authentication key used to gain access to the CBLL API. In order to authenticate with the API, append the access_key parameter to the API's base URL and set it to your access key value.

Base URL:
/https://api.citybylatlong.com
Append your API Access Key:

Here is an example API call illustrating how to authenticate with the ipstack API:

/https://api.citybylatlong.com/?lat=24.000?long=74.00?access_key = YOUR_ACCESS_KEY

Connecting via HTTPS

                                        
All premium subscription plans available for the ipstack API come with support for 256-bit SSL encryption. To connect to the API via HTTPS, simply use the https protocol instead of standard http.

Standard Lookup

                                        
The CBLL's primary endpoint is called Standard Lookup and is used to look up lat-long. To call this endpoint, simply attach any lat-long of India to the API's base URL.

JavaScript (jQuery.ajax)

                                        
Standard Lookup: Find below an example for the Standard Lookup Endpoint using jQuery.ajax.

                                        

                                            
                                                var ip = '134.201.250.155'
                                                var access_key = 'YOUR_ACCESS_KEY';

                                                // get the API result via jQuery.ajax
                                                $.ajax({
                                                url: 'http://api.ipstack.com/' + ip + '?access_key=' + access_key,   
                                                dataType: 'jsonp',
                                                success: function(json) {

                                                // output the "capital" object inside "location"
                                                alert(json.location.capital);

                                                }
                                                });
